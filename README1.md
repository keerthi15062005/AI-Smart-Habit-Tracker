# AI Smart Habit Tracker

An AI-powered full-stack habit tracking web application that analyzes productivity trends and predicts future performance using Machine Learning.

## Features
- User Authentication
- Daily Habit Tracking
- Productivity Analytics Dashboard
- Streak Monitoring
- AI-based Completion Prediction
- Motivational Engine

## Tech Stack
- Flask
- SQLite
- SQLAlchemy
- Chart.js
- scikit-learn

## Installation
pip install -r requirements.txt
python app.py
